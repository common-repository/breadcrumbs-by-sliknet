Breadcrumbs plugin.

== Description ==

Breadcrumbs by slikNET with support schema.org and easy settings


== Installation ==

1) Unzip the archive and put the breadcrumbs_sliknet folder into your plugins folder (/wp-content/plugins/).

OR 

1) Install the plugin through the WordPress plugins screen directly.

2) Activate the plugin from the Plugins menu.

Use the Settings->Breadcrumbs settings screen to configure the plugin

= Usage =

In your theme where you want to display breadcrumbs, just insert this:

`<? breadcrumbs(); ?>`
